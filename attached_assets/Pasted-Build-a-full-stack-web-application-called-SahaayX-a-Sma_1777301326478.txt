=============================================

Database: SQLite (better-sqlite3)
Auth: JWT (30min expiry) + bcrypt passwords
AI Layer 1: Google Gemini 2.0 Flash (via @google/generative-ai SDK) — primary conversational AI
AI Layer 2: Anthropic Claude claude-sonnet-4-20250514 (via @anthropic-ai/sdk) — smart matching engine
Voice: Web Speech API (SpeechRecognition + SpeechSynthesis)

Maps: Leaflet.js + OpenStreetMap tiles
Animations: GSAP 3 + CSS keyframes
Icons: Lucide (CDN unpkg)

=============================================
This must look like a real government/enterprise app, NOT a startup toy.
Study UMANG App, DigiLocker, India Stack, and GOV.UK Design System for reference.

1. NO pure black backgrounds. Use deep navy #0D1B2A as base dark.
2. Every page has a clear HEADER BAR with:
   - Left: SahaayX logo (SVG inline) + Ministry/NGO seal placeholder

   - Right: Language switcher + notifications + user avatar + logout
