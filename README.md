SahaayX- Smart Resource Allocation & Volunteer Coordination Platform

SahaayX is an AI-powered platform that connects community data, volunteers, and authorities into a unified system for real-time coordination. It enables intelligent resource allocation and faster response for social impact initiatives.

Community data, resources, and volunteer efforts are often fragmented, leading to delayed decisions and inefficient coordination. This results in critical support not reaching the right place at the right time.

SahaayX solves this by integrating data, stakeholders, and AI-driven insights into a single platform. It analyzes needs, prioritizes tasks, and matches resources efficiently for coordinated action.

Key Features
🤖 Dual AI Engine (Gemini + Claude)
🗺️ Live Resource Mapping
🧑‍🤝‍🧑 Smart Volunteer Matching
📊 Analytics Dashboard
🎤 Voice-enabled Interaction
🌐 Multi-language Support
🔄 Real-time Data Synchronization

Tech Stack

Frontend: HTML, CSS, JS
Backend: Node.js, Express
Database: SQLite
AI: Gemini + Claude
Others: Chart.js, Leaflet, i18next, Web Speech API

System Architecture 
The platform follows a layered architecture with frontend, backend, AI processing, and database layers working together for real-time coordination.

Usage
Users can upload survey data, analyze needs, view dashboards, and coordinate volunteers through an intuitive interface.
